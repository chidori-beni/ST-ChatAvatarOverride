# 聊天窗口独立头像

### 为 SillyTavern 的每个聊天窗口单独设置角色头像，不影响角色卡原始头像。

适用场景：同一张角色卡开了多个聊天、多人卡不同分支主角不同……总之想不同聊天窗口设置不同的头像。

---

## 安装
在 SillyTavern 扩展页面点「安装扩展」，粘贴以下地址：

```
https://github.com/chidori-beni/ST-ChatAvatarOverride
```

---

## 使用方法

**方式一：在首页聊天列表设置**
鼠标悬停到某个聊天行，点右侧出现的🖼️按钮即可。

**方式二：在聊天窗口内设置**
点输入框左侧的扩展菜单（魔术棒图标），找到「聊天窗口独立头像」。

两个入口效果完全相同，设置后立即同步。

---

## 功能

- 上传本地图片 / 粘贴图片 URL
- 内置裁切工具，支持自由 / 1:1 / 2:3（酒馆标准）/ 16:9 比例，带九宫格参考线
- 设置后首页列表和聊天气泡头像同步更新
- 数据随聊天文件保存，不修改角色卡任何内容
- 清除后自动恢复角色卡原始头像

---

## 注意事项

- 聊天窗口重命名后，窗口头像会被重置（恢复到角色卡默认头像）
- 如果首页列表的🖼️按钮消失了; 或者没有修改聊天窗口名称，但是头像失效了——总之先刷新一下
